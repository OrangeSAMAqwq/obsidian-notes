# 题目：替换数字 (Replace Numbers) 🔢➡️🔤

## 题目链接 🌐  
https://kamacoder.com/problempage.php?pid=1064

---

## 解题思路 🧠

### 题意理解 📋

给定一个只包含 **字母和数字** 的字符串 `s`，要求将其中 **所有数字替换为字符串 `"number"`**，其余字母保持不变。

例如：

- 输入：`a1b2c3`
- 输出：`anumberbnumbercnumber`

题目本身非常直接，只需要遍历字符串并判断每个字符是否为数字。

---

### 解法思路：逐字符判断 + 构建答案字符串 🧩

1. 使用 `cin >> s` 读取输入字符串。
2. 遍历字符串中的每个字符：
   - 如果是 **字母**（`isalpha(s[i])`），直接保留。
   - 否则（也就是数字字符），拼接 `"number"` 到结果字符串中。
3. 最后输出替换后的完整字符串。

---

## 代码实现 💻

```cpp
#include <cstdio>
#include <iostream>
#include <ctype.h>
#include <string>
using namespace std;

int main(){
    string s;
    cin >> s;
    string ans;

    for (int i = 0; i < s.size(); i++){
        if (isalpha(s[i]))
            ans += s[i];        // 字母直接追加
        else
            ans += "number";    // 数字替换为字符串 number
    }

    cout << ans << endl;
    return 0;
}
```

---

## 关键点 🔑

- `isalpha(ch)` 用来判断字符是否为英文字母。
- 题目中不会出现符号，所以除字母外的字符一定是数字，可直接替换。
- 字符串构建时直接用 `+=` 拼接即可。

---

## 边界情况 🚨

1. 字符串全是数字  
   - 例如：`123` → `numbernumbernumber`
2. 字符串全是字母  
   - 例如：`abc` → `abc`
3. 字符串为空（理论上不会输入空）  
4. 字符串长度较大  
   - 需要注意拼接字符串的效率，但 C++ 对于普通规模输入完全足够。

---

## 复杂度分析 ⏱️

- **时间复杂度：O(n)**  
  遍历一遍字符串，对每个字符执行常数操作。

- **空间复杂度：O(n)**  
  新构建的结果字符串最大长度为 `7 * n`（数字替换成 `"number"`）。

- 字符串拼接

实现简单，但非常适合作为入门题目练手。

# 🔗 拓扑排序模板（基于入度的 BFS）

---

## 📌 题目描述

给定一个有向图，共有 `N` 个节点和 `M` 条有向边。  
请你判断该图是否是 **有向无环图（DAG）**，如果是，输出任意一种拓扑排序结果；否则输出 `-1`。

---

## 🧠 算法思路：拓扑排序（Kahn 算法）

拓扑排序适用于 **有向无环图（DAG）**，其核心思想是：

> 每次选择「入度为 0」的点加入序列，并将其所有出边的终点的入度减 1。

如果最后无法处理完所有节点，说明存在环。

---

## ✅ 状态定义

- `NodeIn[i]`：表示第 `i` 个节点的 **入度**
- `grid[i]`：表示从节点 `i` 出发的所有邻接点

---

## 🔁 算法流程

1. 统计所有节点的入度
2. 将所有入度为 0 的点加入队列
3. 依次从队列中取出节点：
   - 放入结果数组
   - 遍历其邻接点，将它们的入度 -1
   - 如果新的入度为 0，再加入队列
4. 如果最终无法处理完所有节点 → 图中有环

---

## 💻 C++ 实现代码

```cpp
#include <cstdio>
#include <iostream>
#include <vector>
#include <queue>
using namespace std;

int main(){
    int N, M;
    cin >> N >> M;

    vector<int> NodeIn(N, 0);                    // 入度数组
    vector<vector<int>> grid(N);                 // 邻接表
    vector<int> ans;                             // 拓扑排序结果

    for (int i = 0; i < M; i++){
        int s, t;
        cin >> s >> t;
        grid[s].push_back(t);                    // 有向边：s → t
        NodeIn[t]++;
    }

    queue<int> q;
    for (int i = 0; i < N; i++){
        if (NodeIn[i] == 0)
            q.push(i);                           // 入度为 0 的点入队
    }

    while (!q.empty()){
        int cnt = q.front(); q.pop();
        ans.push_back(cnt);
        for (int i = 0; i < grid[cnt].size(); i++){
            NodeIn[grid[cnt][i]]--;
            if (NodeIn[grid[cnt][i]] == 0)
                q.push(grid[cnt][i]);
        }
    }

    // 检查是否存在环
    for (int i = 0; i < NodeIn.size(); i++){
        if (NodeIn[i] != 0){
            cout << -1;
            return 0;
        }
    }

    // 输出拓扑序列
    for (int i = 0; i < ans.size(); i++){
        if (i != 0)
            cout << ' ';
        cout << ans[i];
    }

    return 0;
}
```

## 🧪 示例输入输出

## ❌ 示例：存在环

```
输入：
3 3
0 1
1 2
2 0

输出：
-1
```

---

## 🧾 复杂度分析

- 时间复杂度：`O(N + M)`  
- 空间复杂度：`O(N + M)`

---

## 🧠 总结记忆点

- 拓扑排序适用于 **有向无环图（DAG）**
- 拓扑序列长度 < N → 图中有环
- 常用两种方式：
  - BFS（入度统计 + 队列）✅
  - DFS（标记法检测回溯）🚫（本题未使用）

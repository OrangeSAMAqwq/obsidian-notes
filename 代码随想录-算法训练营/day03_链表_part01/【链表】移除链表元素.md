# **移除元素 (Remove Linked List Elements)** ❌
## 题目链接 🌐  
[删除链表中的元素 - LeetCode](https://leetcode.cn/problems/remove-linked-list-elements/)

---

## 解题思路 🧠

### 思路分析 🧩

这道题目要求我们删除链表中所有值为 `val` 的节点。我们可以通过遍历链表，并使用指针操作来删除指定节点。常用的技巧是使用 **虚拟头节点** 来简化链表头部的删除操作，避免边界判断。

### 步骤 📋：

1. **创建虚拟头节点** 🧾：
   - 我们可以创建一个虚拟头节点 `dummy`，使得头节点的删除操作和其他节点的删除操作一样，不需要特殊处理。

2. **遍历链表并删除节点** 🔄：
   - 使用 `pre` 指针指向虚拟头节点，`head` 指针指向链表的当前节点。
   - 遍历链表，检查当前节点的值。如果值等于 `val`，则删除该节点（通过 `pre->next = head->next`）。
   - 否则，`pre` 指针向后移动到当前节点，继续检查下一个节点。

3. **返回结果** 📍：
   - 返回 `dummy->next`，即修改后的链表头。

---

### 时间复杂度 ⏱️：

- **O(n)**，其中 `n` 是链表的长度。我们遍历了整个链表一次。
- 空间复杂度为 **O(1)**，只使用了常数的空间。

---

## 代码实现 💻

```cpp
class Solution {
public:
    ListNode* removeElements(ListNode* head, int val) {
        ListNode* dummy = new ListNode();  // 创建虚拟头节点
        dummy->next = head;
        ListNode* pre = dummy;
        
        while (head) {
            if (head->val == val)  // 如果当前节点的值等于 val，删除该节点
                pre->next = head->next;
            else 
                pre = pre->next;  // 否则，pre 指针向后移动
            head = head->next;  // 遍历下一个节点
        }
        return dummy->next;  // 返回删除后的链表头
    }
};
```

---

## 关键点 🔑：

- **虚拟头节点**：通过虚拟头节点来简化头节点删除操作。
- **指针操作**：使用 `pre` 和 `head` 两个指针来删除指定值的节点。
- **边界情况**：虚拟头节点帮助简化了链表头部的特殊情况处理。

---

## 边界情况 🚨：

1. **链表为空** 🛑：如果链表为空，返回空链表。
2. **所有节点的值都为 `val`** 🔴：如果链表中所有节点都需要删除，最终返回空链表。
3. **链表中没有值为 `val` 的节点** ✅：如果链表中没有需要删除的节点，返回原链表。

---

## 总结 📚

本题通过 **虚拟头节点** 和 **指针操作**，有效地删除了链表中所有值为 `val` 的节点。该方法具有 **O(n)** 的时间复杂度和 **O(1)** 的空间复杂度，适用于链表类问题的经典技巧。

# ✂️ 题目：删除操作得到两个字符串相同（Delete Operation for Two Strings）🧵🧠

---

## 🧠 题目理解

给定两个字符串：

- `word1`
- `word2`

每次操作 **只能删除一个字符**（不能插入、不能替换）。

目标是：

> 通过最少的删除操作，使两个字符串变得 **完全相同**。

---

## 💡 核心思路：字符串 DP（编辑距离的简化版）

这道题本质上是 **编辑距离（Edit Distance）** 的一个特例：

- ❌ 不允许替换
- ❌ 不允许插入
- ✅ 只允许删除

因此，DP 的状态设计和编辑距离非常相似，但转移更简单。

---

## 📐 状态定义

```text
dp[i][j]：表示
word1 的前 i 个字符
和
word2 的前 j 个字符
变成相同字符串所需的最少删除次数
```

---

## 🧱 初始化

### 1️⃣ word2 为空

```text
dp[i][0] = i
```

含义：  
要让 `word1[0..i)` 和空串相同，只能删除 `i` 个字符。

---

### 2️⃣ word1 为空

```text
dp[0][j] = j
```

含义：  
要让 `word2[0..j)` 和空串相同，只能删除 `j` 个字符。

---

## 🔁 状态转移（核心）

考虑当前字符：

- `word1[i-1]`
- `word2[j-1]`

---

### ✅ 情况一：字符相等

```text
word1[i-1] == word2[j-1]
```

无需删除，直接继承之前的状态：

```text
dp[i][j] = dp[i-1][j-1]
```

---

### ❌ 情况二：字符不等

```text
word1[i-1] != word2[j-1]
```

只能通过删除来解决，有两种选择：

1. 删除 `word1[i-1]`
2. 删除 `word2[j-1]`

取最小值：

```text
dp[i][j] = min(
    dp[i-1][j] + 1,   // 删除 word1[i-1]
    dp[i][j-1] + 1    // 删除 word2[j-1]
)
```

---

## 💻 C++ 实现代码

```cpp
class Solution {
public:
    int minDistance(string word1, string word2) {
        int m = word1.size(), n = word2.size();
        vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));

        // 初始化
        for (int i = 1; i <= m; i++)
            dp[i][0] = i;
        for (int j = 1; j <= n; j++)
            dp[0][j] = j;

        // 状态转移
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (word1[i - 1] == word2[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1];
                } else {
                    dp[i][j] = min(
                        dp[i - 1][j] + 1,
                        dp[i][j - 1] + 1
                    );
                }
            }
        }
        return dp[m][n];
    }
};
```

---

## 📊 复杂度分析

- **时间复杂度**：`O(m × n)`
- **空间复杂度**：`O(m × n)`
  - 可优化为一维 DP（滚动数组）

---

## 🧠 核心记忆点 ⭐

```text
只能删除 → 没有 dp[i-1][j-1] + 1（替换）
```

```text
相等：继承
不等：删左 or 删上
```

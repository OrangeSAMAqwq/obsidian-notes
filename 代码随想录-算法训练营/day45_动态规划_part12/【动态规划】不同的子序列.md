# 🧩 题目：不同的子序列（Distinct Subsequences）📚✨

---

## 🧠 题目理解

给定两个字符串：

- `s`：源字符串  
- `t`：目标字符串  

要求统计：  
> **字符串 `t` 在 `s` 中作为子序列出现的个数**

⚠️ 子序列的定义：
- 可以删除 `s` 中的任意字符
- 但不能改变剩余字符的相对顺序

---

## 💡 解题思路：二维动态规划

这是一个 **典型的字符串 DP 计数问题**，核心在于：

> **每一个字符，都有「选」或「不选」两种可能**

---

## 📐 状态定义

```text
dp[i][j]：表示 s 的前 i 个字符中，
          出现 t 的前 j 个字符作为子序列的个数
```

其中：
- `i` ∈ [0, n]
- `j` ∈ [0, m]

---

## 🧱 初始化

### 1️⃣ 空串是任何字符串的子序列

```text
dp[i][0] = 1   （无论 i 是多少）
```

表示：  
`t = ""` 在任意 `s[0..i)` 中都 **恰好有 1 种方式**。

### 2️⃣ 非空 t 不可能由空 s 构成

```text
dp[0][j] = 0   （j > 0）
```

---

## 🔁 状态转移方程（核心）

考虑 `s[i-1]` 和 `t[j-1]`：

### ✅ 情况一：字符相等

```text
s[i-1] == t[j-1]
```

此时有两种选择：

1. **使用 s[i-1] 匹配 t[j-1]**
2. **不使用 s[i-1]**

```text
dp[i][j] = dp[i-1][j-1] + dp[i-1][j]
```

---

### ❌ 情况二：字符不相等

```text
s[i-1] != t[j-1]
```

只能选择：

- 不使用 `s[i-1]`

```text
dp[i][j] = dp[i-1][j]
```

---

## 💻 C++ 实现代码

```cpp
class Solution {
public:
    int numDistinct(string s, string t) {
        int n = s.size(), m = t.size();
        const long long MOD = 1e9 + 7;
        vector<vector<long long>> dp(n + 1, vector<long long>(m + 1, 0));

        // 初始化：空字符串是任何字符串的子序列
        for (int i = 0; i <= n; i++) {
            dp[i][0] = 1;
        }

        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= m; j++) {
                if (s[i - 1] == t[j - 1]) {
                    dp[i][j] = (dp[i - 1][j - 1] + dp[i - 1][j]) % MOD;
                } else {
                    dp[i][j] = dp[i - 1][j] % MOD;
                }
            }
        }
        return dp[n][m];
    }
};
```

---

## 📊 复杂度分析

- **时间复杂度**：`O(n × m)`
- **空间复杂度**：`O(n × m)`
  - 可进一步优化为一维 DP（倒序遍历）

---

## 🧠 记忆要点总结 ⭐

```text
当字符相等：
dp[i][j] = 用它 + 不用它

当字符不等：
dp[i][j] = 不用它
```

一句话记忆：

> **子序列计数问题 =「是否使用当前字符」的选择叠加**

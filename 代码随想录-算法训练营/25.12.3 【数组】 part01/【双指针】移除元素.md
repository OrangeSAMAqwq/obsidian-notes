## 解题思路 🧠

### 双指针法 (Two Pointer Technique) 🔄

双指针法是一种常见的用于解决数组相关问题的技巧。我们可以利用两个指针，一个指向数组的起始位置 `begin`，另一个指向数组的结束位置 `last`，通过不断交换不等于目标值 `val` 的元素，逐步将数组中所有的 `val` 移除。

### 步骤 📝：

1. **初始化指针**：设置两个指针，`begin` 用于从左往右扫描，`last` 用于从右往左扫描。
2. **遍历数组** 🔍：
    - 如果 `nums[begin]` 不等于目标值 `val`，则直接将 `begin` 向右移动。
    - 如果 `nums[begin]` 等于目标值 `val`，则交换 `nums[begin]` 和 `nums[last]`，并将 `last` 向左移动。这样可以将所有的 `val` 移动到数组的末尾。
3. **终止条件** ⏳：当 `begin` 指针超过 `last` 时，说明所有的 `val` 都被移到了数组末尾，此时 `begin` 就是新的数组长度。

### 时间复杂度 ⏱️：

- 时间复杂度为 O(n)，其中 n 是数组的长度。我们只需要一次遍历数组。
- 空间复杂度为 O(1)，因为我们是在原数组上进行操作，没有使用额外的空间。
## 代码实现 💻

```cpp
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        int begin = 0, last = nums.size() - 1;  // 初始化双指针
        while (begin <= last) {
            if (nums[begin] != val) {  // 当前元素不是目标值，继续向右移动
                begin++;
            } else {  // 当前元素是目标值，需要移除
                swap(nums[begin], nums[last]);  // 将目标值与末尾元素交换
                last--;  // 将右指针向左移动
            }
        }
        return begin;  // 返回新的数组长度
    }
};
```

## 关键点 🔑：

- **双指针技术**：通过双指针法交换元素来避免使用额外的空间。
- **元素交换** 🔄：每次遇到目标值时，交换它与数组末尾的元素，并缩小搜索区间。
- **返回新长度** 📏：指针 `begin` 最终会指向新数组的长度。

## 边界情况 🚨

1. **空数组** 🛑：如果输入数组为空，返回的长度应该是 0。
2. **数组中没有目标值** 🚫：如果数组中没有目标值 `val`，则返回数组的原始长度。
3. **数组全是目标值** 🔴：如果数组中的每个元素都等于目标值，则返回 0。

# 并查集应用：冗余边检测（找成环边）

---

## 🧠 问题描述

给定一个无向图，图中包含 `N` 个点和 `N` 条边。  
显然，这其中 **存在一条“冗余边”**（即构成了一个环）。

你需要找出 **这条冗余的边（成环边）**。

---

## 🔧 解法：并查集

- 每输入一条边，判断两个节点是否已经在同一个集合中
- 如果在同一个集合中，说明这条边就是“成环边”
- 否则，就把两个集合合并

---

## ✅ 状态定义

- `father[i]` 表示节点 `i` 的父亲（集合代表元）

---

## 🔁 操作函数

```cpp
// 查找根节点 + 路径压缩
int find(int u, auto &father){
    if (father[u] == u)
        return u;
    return father[u] = find(father[u], father);
}

// 判断是否在同一集合
bool isSame(int u, int v, auto &father){
    return find(u, father) == find(v, father);
}

// 合并两个集合
void joint(int u, int v, auto &father){
    u = find(u, father);
    v = find(v, father);
    if (u == v) return;
    father[u] = v;
}
```

---

## 💻 完整代码

```cpp
#include <cstdio>
#include <iostream>
#include <vector>
using namespace std;

int find(int u, auto &father){
    if (father[u] == u)
        return u;
    return father[u] = find(father[u], father);
}

bool isSame(int u, int v, auto &father){
    return (find(u, father) == find(v, father));
}

void joint(int u, int v, auto &father){
    u = find(u, father);
    v = find(v, father);
    if (u == v)
        return;
    father[u] = v;
}

int main(){
    int N;
    cin >> N;

    // 初始化并查集
    vector<int> father(N + 1, 0);
    for (int i = 1; i <= N; i++){
        father[i] = i;
    }

    int ans1, ans2;

    for (int i = 0; i < N; i++){
        int s, t;
        cin >> s >> t;
        if (isSame(s, t, father)){
            ans1 = s;
            ans2 = t;
        }
        joint(s, t, father);
    }

    cout << ans1 << ' ' << ans2 << endl;
    return 0;
}
```

---

## 🧾 说明

- 一共输入 `N` 条边
- 正常树结构只有 `N-1` 条边
- 因此必定存在一条 **冗余边（成环边）**
- 找出第一条使得两个点已经连通的边即可 ✅

---

## 📌 输出格式示例

```text
输入：
5
1 2
1 3
2 4
3 5
2 5

输出：
2 5
```

说明：边 `(2, 5)` 是多余的，会在已有连接中构成一个环。

## 解题思路 🧠

### 模拟填充（Simulation）🔢

题目要求生成一个 `n x n` 的矩阵，填入 `1` 到 `n^2`，并按 **顺时针螺旋顺序** 进行排列。这种题目很经典，常用**模拟法**来逐步按方向填入数字。

---

### 步骤 📋：

1. **初始化矩阵** 🧾：
   - 创建一个 `n x n` 的二维矩阵，初始值为 `0`。
   - 定义起始点 `(x, y)`，从左上角 `(0, 0)` 开始。

2. **填充数字**：
   - 维护一个计数器 `cnt` 从 `1` 开始，直到 `n * n`。
   - 依次尝试四个方向：
     - 向右 ➡️：`while (y + 1 < n && ans[x][y + 1] == 0)`
     - 向下 ⬇️：`while (x + 1 < n && ans[x + 1][y] == 0)`
     - 向左 ⬅️：`while (y - 1 >= 0 && ans[x][y - 1] == 0)`
     - 向上 ⬆️：`while (x - 1 >= 0 && ans[x - 1][y] == 0)`

3. **边界控制与判断**：
   - 每个方向只填充到当前方向尽头或下一个格子非零为止。
   - 每次填充后更新坐标 `(x, y)` 和计数器 `cnt++`。

---

### 时间复杂度 ⏱️：

- 时间复杂度为 **O(n²)**，每个元素只填入一次。
- 空间复杂度为 **O(n²)**，用于存储结果矩阵。

---

## 代码实现 💻

```cpp
class Solution {
public:
    vector<vector<int>> generateMatrix(int n) {
        vector<vector<int>> ans(n, vector<int>(n, 0));
        int cnt = 1, x = 0, y = 0;
        ans[0][0] = 1;
        while (cnt < n * n) {
            while (y + 1 < n && ans[x][y + 1] == 0) {
                cnt++;
                y++;
                ans[x][y] = cnt;
            }
            while (x + 1 < n && ans[x + 1][y] == 0) {
                cnt++;
                x++;
                ans[x][y] = cnt;
            }
            while (y - 1 >= 0 && ans[x][y - 1] == 0) {
                cnt++;
                y--;
                ans[x][y] = cnt;
            }
            while (x - 1 >= 0 && ans[x - 1][y] == 0) {
                cnt++;
                x--;
                ans[x][y] = cnt;
            }
        }
        return ans;
    }
};
```

---

## 关键点 🔑：

- **方向控制**：使用四个方向顺序执行，分别为 → ↓ ← ↑，并通过判断边界 + 是否为 0 来避免重复。
- **原地填充**：不使用额外空间控制方向，只通过当前格子是否为 0 控制走向。
- **模拟类题型**：是一类非常典型的“模拟移动轨迹”问题，适合练习矩阵操作。

---

## 边界情况 🚨：

1. **n = 1**：返回 `[[1]]` ✅。
2. **n = 0**：返回空矩阵 `[]` 🈳。
3. **大尺寸矩阵**：如 `n = 100` 时，代码仍可正常运行，效率表现良好 💪。

## 总结 📚

本题是典型的“螺旋矩阵填充”模拟题。解法清晰直观，通过控制方向和边界条件即可完成填充。时间复杂度为 **O(n²)**，空间复杂度也为 **O(n²)**，适合作为矩阵模拟类题目的入门练习题。

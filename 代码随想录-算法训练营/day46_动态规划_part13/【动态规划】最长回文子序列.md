# 🪞 题目：最长回文子序列（Longest Palindromic Subsequence）🧠✨

---

## 🧩 题目理解

给定一个字符串 `s`，要求找出其中 **最长的回文子序列长度**。

### ⚠️ 注意区分两个概念

- **子串**：必须连续  
- **子序列**：可以不连续，但相对顺序不能改变  

本题是 **子序列**，不是子串。

---

## 💡 解题思路：区间动态规划（经典）

这是一个 **区间 DP 的经典模板题**，和下面这些题属于同一套路：

- 回文子串
- 最长回文子序列
- 区间合并类 DP

---

## 📐 状态定义

```text
dp[i][j]：
表示字符串 s 在区间 [i, j] 内的
最长回文子序列长度
```

---

## 🧱 初始化与遍历顺序（非常关键）

### 1️⃣ 单个字符一定是回文

```text
dp[i][i] = 1
```

---

### 2️⃣ 遍历顺序

由于状态转移依赖：

```text
dp[i + 1][j - 1]
dp[i + 1][j]
dp[i][j - 1]
```

因此必须保证：

- `i` 从 **后往前**
- `j` 从 **前往后**

正确遍历方式：

```text
i : n-1 → 0
j : i → n-1
```

---

## 🔁 状态转移方程（核心）

考虑区间 `[i, j]`：

---

### ✅ 情况一：两端字符相等

```text
s[i] == s[j]
```

分三种子情况：

#### ① 同一个字符（i == j）

```text
dp[i][j] = 1
```

---

#### ② 相邻字符（j - i == 1）

```text
dp[i][j] = 2
```

---

#### ③ 中间还有字符

```text
dp[i][j] = dp[i + 1][j - 1] + 2
```

---

### ❌ 情况二：两端字符不相等

```text
s[i] != s[j]
```

只能从两种选择中取最大值：

- 不要左端字符
- 不要右端字符

```text
dp[i][j] = max(
    dp[i + 1][j],
    dp[i][j - 1]
)
```

---

## 💻 C++ 实现代码（你的版本）

```cpp
class Solution {
public:
    int longestPalindromeSubseq(string s) {
        int n = s.size();
        vector<vector<int>> dp(n, vector<int>(n, 0));
        int ans = 0;

        for (int i = n - 1; i >= 0; i--) {
            for (int j = i; j < n; j++) {
                if (s[i] == s[j]) {
                    if (i == j)
                        dp[i][j] = 1;
                    else if (j - i == 1)
                        dp[i][j] = 2;
                    else
                        dp[i][j] = dp[i + 1][j - 1] + 2;
                } else {
                    dp[i][j] = max(dp[i + 1][j], dp[i][j - 1]);
                }
                ans = max(ans, dp[i][j]);
            }
        }
        return ans;
    }
};
```

---

## 📊 复杂度分析

- **时间复杂度**：`O(n²)`
- **空间复杂度**：`O(n²)`
  - 可优化为 `O(n)`（一维 DP，倒序更新）

## 🧠 核心记忆点 ⭐（必背）

```text
区间 DP 三件套：
1️⃣ 状态 dp[i][j]
2️⃣ i 从后往前
3️⃣ 转移看两端
```

```text
相等 → 收缩区间 + 2
不等 → 左右取最大
```

## ✅ 总结一句话

> **最长回文子序列的本质：**
>  
> 在区间内不断「向中间收缩」，  
> 相等就加 2，不等就左右取最大。

这道题是 **区间 DP 的代表作之一**，  
非常适合你现在这个阶段反复巩固 👍

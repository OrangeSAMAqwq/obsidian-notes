# 🔍 题目：回文子串（Palindromic Substrings）✨🪞

---

## 🧠 题目理解

给定一个字符串 `s`，要求统计其中 **回文子串** 的个数。

### 回文子串定义

- 子串：必须 **连续**
- 回文：正着读和反着读一样

例如：

```text
s = "aaa"
回文子串有：
"a"（3 个）
"aa"（2 个）
"aaa"（1 个）
总共 6 个
```

---

## 💡 解题思路：动态规划（区间 DP）

这是一个 **典型的区间 DP 问题**，核心思想是：

> 一个字符串是否为回文，取决于：
> - 两端字符是否相等
> - 中间子串是否为回文

---

## 📐 状态定义

```text
dp[i][j]：
表示子串 s[i..j] 是否为回文串
```

- `true`：是回文
- `false`：不是回文

---

## 🧱 初始化与遍历顺序（关键）

### ✔️ 为什么 i 要从后往前遍历？

因为状态转移依赖于：

```text
dp[i+1][j-1]
```

所以必须保证：

- `i+1 > i`
- `j-1 < j`

👉 必须 **先算短区间，再算长区间**

因此遍历顺序是：

```text
i：从 n-1 → 0
j：从 i → n-1
```

---

## 🔁 状态转移方程

考虑子串 `s[i..j]`：

### ✅ 两端字符相等

```text
s[i] == s[j]
```

此时分两种情况：

#### ① 长度 ≤ 2（单字符 / 两字符）

```text
j - i <= 1
```

必然是回文串。

#### ② 长度 ≥ 3

需要中间子串也是回文：

```text
dp[i+1][j-1] == true
```

---

### ✨ 综合条件

```text
if (s[i] == s[j]) {
    if (j - i <= 1 || dp[i+1][j-1]) {
        dp[i][j] = true;
    }
}
```

一旦满足，即可计数。

---

## 💻 C++ 实现代码（你的版本）

```cpp
class Solution {
public:
    int countSubstrings(string s) {
        int n = s.size();
        int ans = 0;
        vector<vector<bool>> dp(n, vector<bool>(n, false));

        for (int i = n - 1; i >= 0; i--) {
            for (int j = i; j < n; j++) {
                if (s[i] == s[j]) {
                    if (j - i <= 1 || dp[i + 1][j - 1]) {
                        dp[i][j] = true;
                        ans++;
                    }
                }
            }
        }
        return ans;
    }
};
```

---

## 📊 复杂度分析

- **时间复杂度**：`O(n²)`
- **空间复杂度**：`O(n²)`

## 🧠 核心记忆点 ⭐

```text
回文串判断：
两端相等 && 中间是回文
```

```text
dp[i][j] 依赖 dp[i+1][j-1]

```

> **回文子串 DP 的本质：**
>  
> 用短区间结果，推出长区间是否为回文。

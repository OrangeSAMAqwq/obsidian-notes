# 题目：删除字符串中的所有相邻重复项 (Remove All Adjacent Duplicates In String) 🔁✂️

## 题目链接 🌐  
https://leetcode.cn/problems/remove-all-adjacent-duplicates-in-string/

---

## 解题思路 🧠

### 题意理解 📋

给定一个仅由小写字母组成的字符串 `s`，重复地删除**相邻且相同的两个字符**，直到不能再删除为止，返回最终剩下的字符串。

例如：

- 输入：`"abbaca"`  
- 删除 `'bb'` → `"aaca"`  
- 再删除 `'aa'` → `"ca"`  
- 输出：`"ca"`

---

### 解法思路：栈的模拟 ✨

我们可以用一个栈（用字符串代替）来**模拟字符入栈与出栈**的过程：

1. 遍历字符串的每个字符；
2. 如果当前字符和栈顶字符相同，则弹出栈顶（即删除一对重复）；
3. 否则，将当前字符压入栈中；
4. 最终栈中剩下的就是结果字符串。

由于我们用 `string ans` 来模拟栈，这样写法简洁、空间连续。

---

## 代码实现 💻

```cpp
class Solution {
public:
    string removeDuplicates(string s) {
        string ans;
        for (int i = 0; i < s.size(); i++) {
            if (ans.empty() || s[i] != ans.back())
                ans += s[i];         // 入栈
            else
                ans.pop_back();     // 弹栈（删除重复）
        }
        return ans;
    }
};
```

---

## 操作示例 🎯

对于输入 `"abbaca"`：

| 操作     | ans 栈状态 | 说明                      |
|----------|-------------|---------------------------|
| 读取 'a' | `'a'`       | 栈空，入栈                 |
| 读取 'b' | `'a','b'`   | 不等于栈顶，入栈           |
| 读取 'b' | `'a'`       | 等于栈顶，弹出              |
| 读取 'a' | `'a','a'`   | 不等于栈顶，入栈           |
| 读取 'c' | `'a','a','c'`| 不等于栈顶，入栈          |
| 读取 'a' | `'c','a'`   | 弹出前两个 'a'，继续       |

结果：`"ca"`

---

## 边界情况 🚨

1. 输入为空字符串 `""`：返回空；
2. 输入无重复项：不变；
3. 输入全部重复如 `"aaaa"`：全被删除；
4. 输入为 `"abba"`：删除 `"bb"` 后变为 `"aa"`，再删除 `"aa"`，最终为空。

---

## 复杂度分析 ⏱️

- 时间复杂度：**O(n)**，每个字符最多入栈出栈一次；
- 空间复杂度：**O(n)**，最多栈中存下所有字符（无重复时）。

---

## 总结 📚

这是一道典型的“**相邻重复消除**”问题，使用 **栈结构（或字符串模拟栈）** 是最优解法，能够高效地处理层层嵌套的删除逻辑。

适合作为练习“栈”思想和“字符串构造”的综合题目 ✅
